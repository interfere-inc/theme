# Interfere

A Radix-derived theme with four variants:

- **Interfere Dark** / **Interfere Light**
- **Interfere Dark Vibrant** / **Interfere Light Vibrant** — wide-gamut (Display-P3)

Colors are generated from a single source of truth and shared across Neovim, VS Code, and terminals. See the [project repository](https://github.com/interfere/theme) for details.

## Install

1. Open the Extensions view (`⇧⌘X` / `Ctrl+Shift+X`)
2. Search for **Interfere**
3. `⌘K ⌘T` / `Ctrl+K Ctrl+T` → pick an Interfere variant

> The theme JSON files in `themes/` are generated — edit the palette/source in the repo, not these files.
